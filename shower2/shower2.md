showermaster is a react mobile app that will be able to do the following. Some features will be added later and should still have presence in the app like a stub or a simple implemantation of the particular feature. Anything marked as future would be for a selection of feature from a hamburger menu. 

## Storge and Data Synch
- store information locally in an sqlite db - have the ability to send db data to an external service (future) with submenu items text, api

## Settings
- User Name (future)
- default measurement units either Imperial (US when app is located in the USA versus using Imperial) or Metric (future)
- synch connection data (future) 

## Primary purpose of app
be able to create information about an installation for shower glass.

Any added edited or deleted data  in the app should be created as a full CRUD app and any changes should always be confirmed
