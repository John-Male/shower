# Shower

A Flutter application with git functionality.

## Getting Started

This is a Flutter project that demonstrates basic app structure with proper git configuration.

### Prerequisites

- Flutter SDK
- Dart SDK
- Git

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/John-Male/shower.git
   cd shower
   ```

2. Install dependencies:
   ```bash
   flutter pub get
   ```

3. Run the app:
   ```bash
   flutter run
   ```

## Project Structure

- `lib/` - Main Dart code
- `test/` - Unit and widget tests
- `android/` - Android-specific code
- `ios/` - iOS-specific code

## Git Configuration

This project includes:
- `.gitignore` for Flutter projects
- Proper file structure for version control
- Metadata for Flutter project configuration
